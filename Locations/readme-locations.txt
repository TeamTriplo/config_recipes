This is a very basic Locations feature that comes with:

1) Content type
2) Two custom fields
3) Three field instances
3) One page view of locations

REQUIREMENTS:

Module: Addressfield - https://backdropcms.org/project/addressfield
Module: Geofield - https://backdropcms.org/project/geofield
Module: Geocoder - https://backdropcms.org/project/geocoder
Module: Geophp - https://github.com/backdrop-contrib/geophp
